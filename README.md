# SITRUS EX

芝浦工業大学の履修登録サイト **SITRUS** を、Chrome 上で見やすく・使いやすくする拡張機能です。  
**大学公式ではありません。** 見た目と操作感の改善のみで、ログインや履修データそのものは公式サイトのままです。

---

## できること

### ログイン画面

- キャンパスの写真が背景でゆっくり切り替わります。
- **認証開始**ボタンを大きく、押しやすく表示します（学生の方は、案内どおり入力なしで始められます）。
- 教職員の方は、**学籍番号の入力**を開いてから入力できます（折りたたみ式）。

### ログイン後の画面

- 画面上部のボタンやサイドバーを、読みやすいレイアウトに整えます。
- 右上の **歯車アイコン** から、**表示テーマ**（雰囲気の違う 3 種類）を選べます。選んだテーマは、この PC の Chrome にだけ覚えさせます。
- 右下に **SITRUS EX のバージョン** が小さく表示されます。

---

## インストール方法

### Chrome ウェブストアから入れる場合

（公開されたら）ストアの案内に従って **Chrome に追加** してください。

### 配布されたフォルダ（ZIP など）から入れる場合

1. PC に展開したフォルダを置きます（中に `manifest.json` がある階層）。
2. Google Chrome でアドレスバーに `chrome://extensions` と入力して開きます。
3. 右上の **デベロッパーモード** をオンにします。
4. **パッケージ化されていない拡張機能を読み込む** を押し、さきほどのフォルダを選びます。

インストール後、**芝浦工業大学の SITRUS**（`sitrus.sic.shibaura-it.ac.jp`）を開くと、自動で見た目が変わります。

## 使い方のヒント

| やりたいこと | 操作 |
|-------------|------|
| テーマを変える | SITRUS にログインしたあと、右上の **歯車** → 好みのテーマをタップ |
| 教職員として学籍番号を入れる | ログイン画面で **学籍番号の入力（教職員向け）** を開いてから入力 |

## プライバシーについて

- 拡張機能は **あなたの PC の中だけ** に、選んだ **テーマ名** を保存します（ブラウザの保存領域）。
- 作者のサーバーへ履修情報や個人情報を送ることはありません。  
  詳しくは [PRIVACY.md](./PRIVACY.md) を参照してください。
